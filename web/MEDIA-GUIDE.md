### 1. Widget Interface Screenshot
**File:** `screenshot-widget.png`
**Recommended Size:** 1200x900px
**Format:** PNG

**What to capture:**
- The floating widget in the top-right corner
- Include and Exclude input fields filled with example keywords
- Show the preset dropdown with a few saved presets
- Status counter showing something like "15 / 20"
- Make sure the Facebook Marketplace page is visible in the background

**Example keywords to use:**
- Include: `gaming, xbox, playstation, console`
- Exclude: `repair, broken, parts`

**Steps:**
1. Open Facebook Marketplace
2. Click "..." on a listing â†’ "Add to multiple places"
3. Fill in the widget with example keywords
4. Load a preset
5. Take screenshot (use Snipping Tool or browser dev tools)
6. Crop to 1200x900px
7. Add subtle drop shadow in photo editor (optional)

### 3. Group Selection Dialog Screenshot
**File:** `screenshot-dialog.png`
**Recommended Size:** 1200x900px
**Format:** PNG

**What to capture:**
- The group prioritization dialog when >20 groups match
- Show the list of groups with checkboxes
- Some groups checked, some unchecked
- The "X matched" and "Y max" badges at the top
- "Confirm Selection" button at the bottom

**Steps:**
1. Set up filters that will match >20 groups
2. Click "Run Checker"
3. Wait for dialog to appear
4. Screenshot the dialog
5. Ensure group names are visible but privacy is maintained (you can blur user-specific info)

## ðŸŽ¥ Product Demo Video

**File:** `demo.mp4`
**Recommended Duration:** 30-60 seconds
**Recommended Resolution:** 1920x1080 (1080p)
**Max File Size:** 10-15 MB (compress if needed)
**Format:** MP4 (H.264 codec)

### Video Script (45 seconds)

**0:00-0:08** - Opening
- Screen recording of Facebook Marketplace page
- Cursor clicks on "..." on a listing
- Selects "Add to multiple places"
- Text overlay: "Posting to Facebook Groups?"

**0:08-0:18** - Show the problem
- Facebook shows hundreds of groups
- Cursor scrolls through the long list
- Text overlay: "Stop wasting time scrolling!"

**0:18-0:28** - Show the solution
- Widget appears in corner
- Cursor types keywords: "gaming, xbox"
- Clicks "Run Checker"
- Groups automatically select
- Text overlay: "Auto Groups Checker finds relevant groups instantly"

**0:28-0:38** - Show features
- Quick cut showing:
  - Preset dropdown
  - Status counter changing
  - Multiple groups being checked
- Text overlay: "Save presets, track status, pick priorities"

**0:38-0:45** - Call to action
- Widget looking neat in corner
- Text overlay: "Install Auto Groups Checker - It's Free!"
- Show Chrome logo or "Add to Chrome" button
- Fade to logo

### Recording Tools

**Free:**
- **OBS Studio** (Windows/Mac/Linux) - Best quality, open source
- **ShareX** (Windows) - Easy and powerful
- **Loom** (Web-based) - Quick and simple
- **Screencast-O-Matic** (Web) - Browser-based

**Paid:**
- **Camtasia** - Professional editing
- **ScreenFlow** (Mac) - Easy to use
- **Snagit** - Simple and effective

### Video Recording Tips

1. **Clean your browser:**
   - Close unnecessary tabs
   - Hide bookmarks bar
   - Use incognito mode to avoid personal info

2. **Prepare the demo:**
   - Practice the workflow 2-3 times
   - Have example listings ready
   - Pre-fill some presets

3. **Recording settings:**
   - 1920x1080 resolution (or your screen's native res)
   - 30 FPS minimum (60 FPS better)
   - Cursor highlighting ON
   - Record system audio if adding voiceover

4. **Editing:**
   - Speed up slow parts (1.5x-2x)
   - Add smooth transitions
   - Include text overlays for clarity
   - Add background music (royalty-free)

5. **Compression:**
   - Use HandBrake (free) or similar
   - Target: < 10MB file size
   - H.264 codec, MP4 container
   - Quality: ~80% (CRF 23-28)

## ðŸŽ¨ Design Tips

### Color Palette
Use the extension's brand colors:
- Primary Green: `#10b981`
- Dark Green: `#059669`
- Dark: `#111827`
- Light Gray: `#f9fafb`
- Medium Gray: `#6b7280`

### Fonts
- **Headlines:** Inter, SF Pro Display, Segoe UI (bold)
- **Body:** Inter, SF Pro Text, Segoe UI (regular)

### Screenshot Quality
1. Use 2x or Retina display if available
2. Clean UI (no clutter)
3. Good contrast and readability
4. Compress with TinyPNG or ImageOptim
5. Use PNG for screenshots (better quality)
6. Use JPG for photos/complex images (smaller size)
